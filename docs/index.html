01

What it does

Not a template generator. A social-media manager that does the work and shows you everything.
🖱️

It does the clicking

Using the Claude Chrome extension, it drives LinkedIn for you — headline, About, experience, skills, banner. Prefer to stay hands-on? It hands you copy-paste-ready text with a "where this goes" guide instead. Your call, made on the first screen.

🎯

Written for the right reader

A 7-question interview decides who your profile speaks to — recruiter, client, or peer. Every line is then written for that one person.

✍️

No clichés, no AI-slop

A built-in filter blocks "passionate", "results-driven", "synergy", and the em-dash rhythm that screams machine-written. It reads like you.

You answer honest questions and approve changes. It does the heavy lifting.
STAGE / 01

Welcome & setup

Re-checks everything went through. Produces a before/after report.

STAGE / 08

Launch

Offers case-study pages, walkthrough videos, and a draft of your first post.

03

Up and running in three steps

You need the Claude app and the Claude Chrome extension. That's it.
1
Install the Claude Chrome extension and sign in. (Full guide in the repo.)
2
Clone the repo and copy the three skill folders into your Claude skills directory:
bash
git clone https://github.com/ShayantoDutta/linkedin-revamp.git
cd linkedin-revamp
cp -r linkedin-revamp linkedin-revamp-notion linkedin-revamp-walkthrough ~/.claude/skills/
PowerShell
git clone https://github.com/ShayantoDutta/linkedin-revamp.git
cd linkedin-revamp
Copy-Item -Recurse linkedin-revamp,linkedin-revamp-notion,linkedin-revamp-walkthrough $env:USERPROFILE\.claude\skills\
3
Restart Claude, then type /linkedin-revamp — or just say "fix my LinkedIn".

06

Questions, answered

The honest version. No marketing.
Do I need to know how to code?
No. Every question is plain English and the skill does the technical work. The only mildly technical step is the one-time install (copy-paste the commands above).
Is it safe to let it edit my LinkedIn?
It backs up your profile first, previews every change, and asks before anything irreversible. You can type abort at any moment. It never touches your password, messages, or billing. The full threat model is in the repo's SECURITY.md.
What if I don't want it clicking around my browser?
On the first screen you choose what to switch on. Turn the browser automation off and the skill instead writes everything as copy-paste-ready text with a "where to paste this" guide.
Will my connections get notified on every edit?
No. The skill turns off "Share profile updates with your network" before it edits anything, and restores your original setting when it's done.
Does it make things up?
No. It only reframes what's already in your CV and profile. If a project has no real number yet, it says so honestly rather than inventing one.
What does it cost?
The skill is free and open source (MIT). You need a Claude account and the Chrome extension.
It's brand new — has it been battle-tested?
It's a fresh v1.0 release. Built carefully and reviewed, but not yet run across many profiles in the wild. Treat the first run as a careful one — and the full backup means anything can be rolled back.
